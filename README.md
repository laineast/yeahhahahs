# VerseVault

VerseVault is a simple and aesthetic verse study website made with HTML, CSS, and JavaScript.

It helps users:
- study verses daily
- build consistency
- maintain streaks
- stay motivated through repetition and discipline

## Features

- Daily verse display
- Streak counter
- Study button interaction
- Minimal dark-themed UI
- Beginner-friendly code

## Technologies Used

- HTML
- CSS
- JavaScript

## Future Plans

- Add real daily streak system
- Add streak freezes
- Save progress with local storage
- Multiple verse collections
- Login system
- Database integration
- Different themes/designs
- Verse memorization mode
- Mobile responsiveness improvements

## Project Goal

This project was created to turn daily verse studying into a consistent and encouraging habit through a calm and visually pleasing environment.
